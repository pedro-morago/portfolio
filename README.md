# Portfolio · Pedro Morago

Web personal de portfolio de QA Engineer. Página estática sin dependencias: HTML, CSS y JavaScript vanilla.

Disponible en dos idiomas con selector en la navegación:
- Español: `index.html` → `https://pedro-morago.github.io/portfolio/`
- Inglés: `en/index.html` → `https://pedro-morago.github.io/portfolio/en/`

## Proyectos destacados

- **[CI Shard Advisor](https://github.com/pedro-morago/ci-shard-advisor)** — Paraleliza Cypress en CI de forma óptima, sin Cypress Cloud.
- **[Screen Recorder · QA mode](https://github.com/pedro-morago/grabador-pantalla)** — Extensión de Chrome para reportar bugs con vídeo, consola, HAR y pasos sincronizados.
- **Spins Trainer** — En desarrollo.

## Desarrollo local

No hay build: abre `index.html` en el navegador, o sirve la carpeta con:

```bash
npx serve .
```

## Publicar en GitHub Pages

1. Ve a **Settings → Pages** en este repositorio.
2. En *Source*, elige la rama principal y la carpeta `/ (root)`.
3. La web quedará publicada en `https://pedro-morago.github.io/portfolio/`.
